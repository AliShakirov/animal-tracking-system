# Разработка модуля учета животных для фермерского хозяйства

## Описание проекта

Проект представляет собой модуль учета животных для фермерского хозяйства. Он позволяет отслеживать информацию о животных (возраст, порода, состояние здоровья) и автоматизировать процесс учета. Модуль может быть использован в небольших и крупных фермерских хозяйствах для улучшения управления животноводством.

### Основные функции:
- Учет животных по категориям.
- Отслеживание данных о здоровье животных.
- Вывод статистики по состоянию животных.
